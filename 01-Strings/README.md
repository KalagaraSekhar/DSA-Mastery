# Strings

> Strings are one of the most frequently asked topics in coding interviews and competitive programming. They are immutable sequences of characters in many programming languages and form the basis of text processing.

---

# Learning Objectives

After completing this module, you should be able to:

- Understand how strings are represented.
- Traverse and manipulate strings efficiently.
- Perform common string operations.
- Analyze time complexities of string operations.
- Solve beginner-level competitive programming problems.
- Build a strong foundation for advanced algorithms like KMP, Rabin-Karp, Trie, Suffix Array, etc.

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access Character | O(1) |
| Traversal | O(n) |
| Find Length | O(n) (C) / O(1) (C++) |
| Comparison | O(n) |
| Concatenation | O(n) |
| Reverse | O(n) |
| Substring | O(n) |

# Problem Solving Pattern

While solving string problems, always think in this order:

1. Read Input
2. Traverse Character by Character
3. Identify the Required Pattern
4. Perform Necessary Operations
5. Print the Result

# Key Takeaways

- Strings are simply arrays of characters.
- Most string problems involve traversal.
- Learn to use two pointers when possible.
- Pay attention to ASCII values.
- Practice character manipulation frequently.
- Think about edge cases:
  - Empty string
  - Single character
  - All same characters
  - Uppercase vs Lowercase
